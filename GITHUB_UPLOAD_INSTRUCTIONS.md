# GitHub Upload Instructions

Upload the contents of this folder to:

https://github.com/Dustin-Baldwin/Simpl-Ops

## Recommended GitHub Web Upload

1. Open the repo in GitHub while signed in.
2. Click **Add file**.
3. Click **Upload files**.
4. Drag the contents of this folder into GitHub.
5. Commit with this message:

```text
Add final public Simpl Ops portfolio replica
```

## Upload These

- `README.md`
- `.gitignore`
- `requirements.txt`
- `docs/`
- `diagrams/`
- `examples/`
- `tests/`

## Do Not Upload

Do not upload the original Simpl Ops ZIP, production `app.py`, live URLs, Make.com blueprints, Stripe docs, real schemas, dashboard tokens, client data, private prompts, or production secrets.
